# 🚀 AURORA — Deep Space Mission

An interactive **deep-space mission simulation** built with **HTML5 Canvas, CSS, and JavaScript**.

Navigate the photon-class spacecraft **Aurora** through asteroid fields, comet storms, dark zones, and cosmic nebulae on a journey to **Kepler-452b**, a potentially habitable planet located **1,400 light-years from Earth**.

This project demonstrates **real-time rendering, animation loops, procedural graphics, and interactive controls directly in the browser.**

## ✨ Features

🚀 Interactive spacecraft navigation
🌠 Real-time starfield rendering using the **Canvas API**
🪐 Procedurally generated planets and obstacles
☄️ Dynamic comet storms and asteroid fields
🌌 Animated nebula visual effects
🛰 Mission phases with narrative storytelling

### 📊 Mission HUD

Displays real-time spacecraft information:

* Hull integrity
* Velocity
* Sector
* ETA
* Mission phase

### 🎮 Control Systems

* Keyboard controls
* Touch buttons
* Swipe gestures
* Pinch gesture for warp drive

📱 Fully responsive for **desktop and mobile devices**.

## 🌍 Mission Story

Year **2157**.

Humanity launches the photon-class vessel **Aurora** to explore a new habitable world: **Kepler-452b**.

The crew enters cryogenic sleep while Aurora travels across interstellar space, navigating cosmic hazards and deep-space anomalies.

Your mission:

Guide Aurora safely through space and reach humanity’s potential new home.
